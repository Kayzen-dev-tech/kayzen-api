ü§ñ AI API

Poe AI Chat

POST /api/ai/poe

Endpoint ini digunakan untuk chat dengan AI menggunakan Poe.com

Request Body (JSON):

Parameter Type Status Deskripsi
message string Required Pesan yang ingin dikirim ke AI
model string Optional Model AI (default: capybara). Pilihan: gpt-3.5-turbo, claude, capybara, dll
apiKey string Required API Key dari Poe.com
üí° Cara Mendapatkan API Key Poe.com:
  1. Buka https://poe.com/api_key
  2. Login dengan akun Poe Anda
  3. Generate API Key baru
  4. Copy API Key dan gunakan di request

üí° Tips & Best Practices

1. Rate Limiting: Gunakan API dengan bijak. Jangan melakukan spam request.

2. Error Handling: Selalu handle error response dengan baik di aplikasi Anda.

3. URL Encoding: Pastikan URL parameter di-encode dengan benar.

4. Cache Response: Cache response untuk mengurangi jumlah request yang sama.

5. Update: Follow channel WhatsApp untuk mendapatkan info update terbaru.

üåê Code Examples

JavaScript (Fetch API)

const url = 'https://api.kayzen.com/api/mxdrop/download';
const params = new URLSearchParams({
    url: 'https://mxdrop.to/xxxxx'
});

fetch(`${url}?${params}`)
    .then(response => response.json())
    .then(data => {
        if (data.success) {
            console.log('Download URL:', data.data.downloadUrl);
        }
    })
    .catch(error => console.error('Error:', error));

Python (Requests)

import requests

url = 'https://api.kayzen.com/api/mxdrop/download'
params = {
    'url': 'https://mxdrop.to/xxxxx'
}

response = requests.get(url, params=params)
data = response.json()

if data['success']:
    print('Download URL:', data['data']['downloadUrl'])

Python (Requests) - Pinterest Search

import requests
import time

url = 'https://api.kayzen.com/api/pinterest/search'
params = {
    'query': 'frieren',
    'limit': 5
}

response = requests.get(url, params=params)
data = response.json()

if data['success']:
    print(f'Found {data["totalResults"]} results')
    for pin in data['data']:
        print(f'Title: {pin["title"]}')
        print(f'Image: {pin["imageUrl"]}')
        print('---')
        time.sleep(2)

PHP (cURL)

$url = 'https://api.kayzen.com/api/mxdrop/download';
$params = http_build_query([
    'url' => 'https://mxdrop.to/xxxxx'
]);

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url . '?' . $params);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);

if ($data['success']) {
    echo 'Download URL: ' . $data['data']['downloadUrl'];
}
