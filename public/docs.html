Downloader & Search

GET /api/tiktok?url=...
Download TikTok video without watermark (TikWM).
GET /api/mxdrop?url=...
MXDrop video data fetcher.
GET /api/pinterest?q=query
Search Pinterest and return 5 image links.

Artificial Intelligence (5 Features)

GET /api/ai/chat?text=hello
Simple AI Chat response.
GET /api/ai/image?prompt=cat
Generate image from text (Pollinations).
GET /api/ai/summary?text=longtext
Summarize text.
GET /api/ai/translate?text=hello&lang=id
Translate text to specific language.
GET /api/ai/code?query=function
Generate code snippets.
