üìå Pinterest Search API

Search Pinterest

GET /api/pinterest/search

Endpoint ini digunakan untuk mencari gambar, video, dan pins dari Pinterest

Query Parameters:

Parameter Type Status Deskripsi
query string Required Kata kunci pencarian (contoh: frieren, anime, nature)
cookie string Optional Cookie dari Pinterest (untuk hasil lebih baik)
csrftoken string Optional CSRF Token dari Pinterest
üí° Tips: Untuk hasil yang lebih baik, Anda bisa menambahkan cookie dan csrftoken dari browser Anda:
  1. Buka Pinterest di browser dan login
  2. Tekan F12 untuk membuka Developer Tools
  3. Buka tab "Application" ‚Üí "Cookies" ‚Üí "https://id.pinterest.com"
  4. Copy value dari cookie "_pinterest_sess" dan "csrftoken"
  5. Gunakan sebagai parameter optional di API

üí° Tips & Best Practices

1. Rate Limiting: Gunakan API dengan bijak. Jangan melakukan spam request.

2. Error Handling: Selalu handle error response dengan baik di aplikasi Anda.

3. URL Encoding: Pastikan URL parameter di-encode dengan benar.

4. Cache Response: Cache response untuk mengurangi jumlah request yang sama.

5. Update: Follow channel WhatsApp untuk mendapatkan info update terbaru.

üåê Code Examples

JavaScript (Fetch API)

const url = 'https://api.kayzen.com/api/mxdrop/download';
const params = new URLSearchParams({
    url: 'https://mxdrop.to/xxxxx'
});

fetch(`${url}?${params}`)
    .then(response => response.json())
    .then(data => {
        if (data.success) {
            console.log('Download URL:', data.data.downloadUrl);
        }
    })
    .catch(error => console.error('Error:', error));

Python (Requests)

import requests

url = 'https://api.kayzen.com/api/mxdrop/download'
params = {
    'url': 'https://mxdrop.to/xxxxx'
}

response = requests.get(url, params=params)
data = response.json()

if data['success']:
    print('Download URL:', data['data']['downloadUrl'])

Python (Requests) - Pinterest Search

import requests

url = 'https://api.kayzen.com/api/pinterest/search'
params = {
    'query': 'frieren'
}

response = requests.get(url, params=params)
data = response.json()

if data['success']:
    print(f'Found {data["totalResults"]} results')
    for pin in data['data']:
        print(f'Title: {pin["title"]}')
        print(f'Image: {pin["imageUrl"]}')
        print('---')

PHP (cURL)

$url = 'https://api.kayzen.com/api/mxdrop/download';
$params = http_build_query([
    'url' => 'https://mxdrop.to/xxxxx'
]);

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url . '?' . $params);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);

if ($data['success']) {
    echo 'Download URL: ' . $data['data']['downloadUrl'];
}
