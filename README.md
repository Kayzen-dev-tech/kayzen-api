## How to Install Locally

1. Clone this repository.
2. Run `npm install` to install dependencies.
3. Add your images (slide1.jpg... slide15.jpg, etc.) to the `public` folder.
4. Run `npm start`.
5. Open `http://localhost:3000`.

## Deployment

This project is optimized for **Vercel**.
1. Install Vercel CLI or link GitHub to Vercel Dashboard.
2. Deploy.
